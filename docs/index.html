Termux User Repository PyPI Index

Unofficial pre-compiled Python wheels for Termux on aarch64 (Android 7+).

Use this index with pip/uv:

pip install --upgrade pip 
# or use uv: uv pip install --upgrade pip

# Install packages:
pip install --extra-index-url https://guspix.github.io/pypi/ SomePackage
# OR with uv:
uv pip install --extra-index-url https://guspix.github.io/pypi/ SomePackage

Note: This index provides wheels originally built for linux_aarch64, relabeled and served as android_24_aarch64 for compatibility with tools like uv in Termux. Compatibility is not guaranteed for all packages.
